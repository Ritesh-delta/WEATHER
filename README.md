# 🌦️ Weather App

A simple and beautiful weather application built with **Python 🐍** and **Tkinter 🖥️** that fetches real-time weather data using the OpenWeatherMap API ☁️

---

## ✨ Features

✅ Search weather by city name  
✅ Real-time weather updates 🌍  
✅ Displays:

- 🌡️ Temperature
- 🔥 Min & Max Temperature
- 💧 Humidity
- 🌀 Pressure
- 🌅 Sunrise Time
- 🌇 Sunset Time
- ☁️ Weather Condition

✅ Modern Dark UI 🌙
